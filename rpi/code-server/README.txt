Web-based Visual Studio Code with Python 3 pre-installed alongside
MicroPython tools and VS Code extensions.

Based on the work of:
* https://fleet.linuxserver.io/image?name=linuxserver/code-server
* https://github.com/cdr/code-server
* https://github.com/micropython/micropython/tree/master/tools/mpremote
* https://github.com/espressif/esptool
* https://github.com/DavesCodeMusings/mpremote-vscode
* https://github.com/DavesCodeMusings/esptool-vscode


In here you will find:
* The Dockerfile used by GitHub Actions to build the container.
* The Docker Compose used to run the container.

The container image is only compatible with ARM64 (Raspberry Pi 3
and up) or x86_64 (AMD/Intel PC)

Available from both Docker Hub and GitHub Container Resgistry.

davescodemusings/code-server-python:latest
ghcr.io/davescodemusings/code-server-python:latest
